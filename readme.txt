   - exisiting-data -- contains test data used by unit tests
      in particular implementations
 
   - interop-data -- contains testdata produced by identical parameters
      with different versions of keyczar

   - special-case -- contains special case data that would be difficult
      to produce to tests against

   - make-interop-data -- Vagrant box, with checkout, build and test scripts,
      to create new interop-data
